## 50 Proyectos en Python

Repositorio donde desarrollo 50 proyectos para mejorar mis habilidades en Python, reforzar fundamentos y construir un portafolio sólido.

Cada proyecto incluye:
- Código fuente
- Versiones evolutivas (v1, v2, etc.)
- Mejoras progresivas
- Buenas prácticas
