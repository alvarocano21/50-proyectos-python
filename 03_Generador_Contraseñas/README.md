# Generador de Contraseñas

Generador personalizado de contraseñas seguras.

## Funcionalidades

- Selección de cantidad de letras, números y símbolos
- Mezcla aleatoria con random.shuffle()
- Validación de longitud mínima recomendada
- Conversión de lista a string con join()

## Versiones

### v1
- Generación básica personalizada

### v2 (pendiente)
- Manejo avanzado de errores
- Validación estricta de seguridad

## Conceptos practicados

- Listas
- Bucles for
- random.choice()
- random.shuffle()
- join()
- try/except
