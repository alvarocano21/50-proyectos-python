# Piedra, Papel o Tijera

Juego clásico contra la computadora.

## Versiones

### v1
- Elección aleatoria con random.choice()
- Lógica condicional para determinar ganador

### v2
- Bucle infinito hasta escribir "salir"
- Marcador acumulativo
- Validación de entrada
- Manejo de errores comunes

## Conceptos practicados

- while True
- break / continue
- Operadores lógicos (and / or)
- Variables acumuladoras
- Módulo random
